## 复现注意事项

- API-backed evaluators 要求 backend 运行在指定 `--base-url`。
- 本地 storage root 由 `STORAGE_ROOT` 决定。
- DeepSeek/LLM 评测默认不需要真实 key；真实 key 只能放本地 `.env` 或环境变量，并且必须被脱敏。
- Optional Docling/Unstructured 缺失时，不影响 backend 主验证。
- Package verification 与 strict semantic validation 是两个不同指标。
- 当前没有独立 production shadow/blind gold corpus；不能用 `not_run` 报告宣称 0.85。
