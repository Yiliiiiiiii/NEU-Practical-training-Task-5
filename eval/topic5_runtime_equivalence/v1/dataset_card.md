# Topic 5 Runtime Equivalence v1

This evaluator runs the same semantic UIR, Schema, mapping rules, content
organization, metadata template, and execution options through two engine
contexts representing inline and registered execution.

Cases: announcement, event notice, general document, meeting document, and
policy document. Operational identifiers, timestamps, paths, and durations are
excluded from semantic hashes. The result demonstrates convergence of Topic 5
adapters over these public fixtures; it is not a claim about other topics or
arbitrary external schemas.
